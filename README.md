## 주요기능
<br>
- 서비스 설명 : 모두가 학습의지가 샘솟을 수 있는 스터디 플랫폼
- 주요 기능 :
	- WebRTC를 통한 실시간 화상 플랫폼
	- Teachable Machine pose을 통한 자세를 인식하여 학습기록
	- 공부시간을 활용하여 사용자들의 랭킹시스템
